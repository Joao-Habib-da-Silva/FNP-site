P

P é conhecido por ser a tag paragrafo usada no html, não só html como em algumas funções feitas pelo W3C community, como por exemplo arquivos MD.

Isso aqui é um paragrafo

<p> Isso aqui é um paragrafo </p>

H1 E H6

H1 até H6 são tags conhecidas como cabeçalho, como se fosse um titulo da página HTML, elas podem ir de seu maior tamanho que é a tag h1, mas também podem diminuir, sendo h2 menor qur h1, e assim se repetindo até o h6, o menor.

Temos uma tag h1!

<h1> Temos uma tag h1! </h1>

IMG

IMG tag usada para colocarmos imagens, pode ser de todos os tipos, menos SVG, já que o Svg tem uma própria tag que podemos manipular ela com seus próprios atributos

IMG tem um atributo principal que seria o src, onde colocamos o diretório que a imagem está em seu computador ou dispositivo, recomendo que coloque a imagem no mesmo diretório que o arquivo HTML que você use.

< img src="imagem.jpg" > < /img >

A href

A tag A HREF serve para fazermos links, href é o atributo que colocamos o link, o link pode tanto ser uma ID do documento, pode ser outro arquivo, ou um link externo, e até mesmo um arquivo para baixar

Esse é o HREF
< img src="imagem.jpg" > < /img >

UL e LI

  • Primeiro item
  • Segundo item
  • Terceiro item
  • Quarto item
< ul > <li >Primeiro item </li > <li >Segundo item < /li> <li >Terceiro item < /li> <li >Quarto item < /li> < /ul>

Audio

Tag audio já diz pelo nome, ela serve para colocarmos audio, seus atributos são importantes, controls serve para disponibilizar a tela de botões para manusear o aúdio, loop serve para saber se o aúdio vai repetir depois de seu termino autoplay é o atributo que nos possibilita que o aúdio inicie na hora que estiver preparado ao abrir a página, muted serve para deixar o aúdio mutado quando iniciado. A parte source src serve para colocarmos o aúdio

< audio controls muted loop autoplay > <source src="" > </audio >

DIV

Div são divisórias tem como seu principal atributo a id e class, id e class podem ser usadas em todas as tags que são elementos dentro da tag body, imagine eles como nomes, só que tem uma diferença, o ID é uma div, ou melhor para entender, uma pessoa, com nome único, só ele tem esse nome, enquanto o Class é um nome usado por outras tags... pessoas ops! Por enquanto iremos usar o CSSinline apenas para entender a diferença, você não precisa entender de CSS, não ainda. Então em conceito técnico, divisórias nos permite com que fazemos configurações estilizadas, além de trazer um código mais legivel e clean, e sobre id e class, id só pode ser usado em uma situação, class pode ser usado várias vezes, class poderia também ser usada uma vez, mas isso é antiprofissional, não recomendo!

<div id="CLAUDIO"style="width:100px; height: 100px; background-color: green;" > < /div> < div class="ROBERTO" style="width: 100px; height: 100px; background-color: yellow;" > < /div> < div class="ROBERTO" style="width: 100px; height: 100px; background-color: yellow;" > < /div>

Header

Header, não tem como eu fazer o exemplo do header, afinal, nosso site já tem um! Entretanto, posso explicar como funciona o header. O header ele faz parte de um grupo de tags que parecem divs, mas elas são especificas para certas situações, o header serve para fazer o cabeçalho, a parte de cima como menu que a maioria dos sites tem.

FOOTER

Assim como o header, ele também é um tipo de div, só que para situações especificas, o footer, ele serve para fazer o pé da página, aquela parte que os sites tem que tem o contato, as redes sociais do site, e politicas do site ou da empresa que fez ele

SPAN

SPAN, não tem uma função especifica, ele é como uma divisória também, só que ele é usado para colocar como um tipo de divisória em tags fraseadas, quando queremos mudar a cor de algo por exemplo, ou seja, ele é util para o style do CSS

Aqui nós temos um texto esse aqui é o span

<p >Aqui nós temos um texto <span style="color:red;"> </span > </p >

HEAD

Outra tag que não conseguimos exemplos, mas sim a sua explicação, a tag HEAD serve para deixarmos as tags padrões com as configurações do HTML, como por exemplo temos o title, que é o titulo da página, temos o link que é o qual usamos para colocar o link do css, o meta que coloca as configurações como por exemplo a viewport e o name serve para deixarmos possivel a responsividade, onde a tag vai ser adaptar de acordo com a largura do dispositivo

Esses são os mais usados

<input type="text" > <input type="number" > <input type="range" > <input type="checkbox" > <input type="radio" > <input type="button" >

IFRAME

Iframe, usado para incorporar alguma página html em um site, ou o conteúdo de um outro site. Ex: Youtube, o youtube ao apertar o botão direito do mouse, temos como opção, copiar o codigo de incorporação, onde pegamos o vídeo

<iframe width="935" height="526" src="https://www.youtube.com/embed/3_JE76PKBWE" title="How to create a 'What I Use' blog with Astro and Tailwind CSS (at a hotel lounge)" frameborder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share" referrerpolicy="strict-origin-when-cross-origin" allowfullscreen> </iframe>
